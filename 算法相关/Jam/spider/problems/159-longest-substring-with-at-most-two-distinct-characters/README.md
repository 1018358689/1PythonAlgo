# [Longest Substring with At Most Two Distinct Characters][title]

## Description

给定一个字符串 ** _s_** ，找出  **至多  **包含两个不同字符的最长子串 **_t_ **，并返回该子串的长度。

**示例 1:**
            **输入:** "eceba"    **输出:** 3    **解释: _t_** 是 "ece"，长度为3。    

**示例 2:**
            **输入:** "ccaabbb"    **输出:** 5    **解释: _t_** __ 是 "aabbb"，长度为5。
