# [Number Of 2s In Range LCCI][title]

## Description

编写一个方法，计算从 0 到 n (含 n) 中数字 2 出现的次数。

**示例:**
            **输入:** 25    **输出:** 9    **解释:** (2, 12, 20, 21, 22, 23, 24, 25)(注意 22 应该算作两次)

提示：

  * `n <= 10^9`


**Tags:** Recursion, Math, Dynamic Programming

**Difficulty:** Hard
