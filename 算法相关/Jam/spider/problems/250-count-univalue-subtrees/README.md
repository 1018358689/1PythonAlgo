# [Count Univalue Subtrees][title]

## Description

给定一个二叉树，统计该二叉树数值相同的子树个数。

同值子树是指该子树的所有节点都拥有相同的数值。

**示例：**
            **输入:** root = [5,1,5,5,5,null,5]                      5                 / \                1   5               / \   \              5   5   5        **输出:** 4    


**Tags:** Tree, Depth-First Search, Binary Tree

**Difficulty:** Medium
